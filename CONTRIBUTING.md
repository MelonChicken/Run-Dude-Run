# How to Contribute

Thank you for your interest in contributing to **Run-Dude-Run**.
Please follow the steps below to submit changes.

---

## 1. Create a Branch

* Use a feature branch based on `main`.

```
git checkout -b feature/your-feature-name
```

---

## 2. Make Changes

* Keep commits clear and meaningful.
* Follow the existing code style.
* Test your changes before committing.

---

## 3. Commit Your Work

```
git commit -m "Describe your change"
```

---

## 4. Push and Open a Pull Request

```
git push origin feature/your-feature-name
```

Then open a Pull Request (PR) to `main` on GitHub.

---

## 5. Pull Request Guidelines

* Explain what your change does using PR Templates.
* Keep PRs small and focused.
* The PR will be reviewed before merging.

---

## 6. Code Review

* Be open to feedback.
* Make required updates and push again; the PR will be automatically updated.

---

## 7. License

By submitting a contribution, you agree that your code will be released under this project's license.
